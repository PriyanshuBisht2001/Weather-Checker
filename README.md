# 🌦️ Weather Forecast Web App

![Node.js](https://img.shields.io/badge/Node.js-18+-green)
![Express](https://img.shields.io/badge/Express.js-Backend-black)
![API](https://img.shields.io/badge/API-OpenWeather-blue)
![License](https://img.shields.io/badge/License-MIT-yellow)
![Status](https://img.shields.io/badge/Project-Active-success)

A simple and responsive **Weather Forecast Web Application** built using **Node.js, Express, and the OpenWeather API**.
Users can enter a city name and instantly get the current temperature and weather condition.

This project demonstrates **API integration, backend routing, and form handling** in a clean and beginner-friendly way.

# ✨ Features

* 🌍 Search weather by **city name**
* 🌡️ Displays **current temperature**
* ☁️ Shows **weather condition**
* ⚡ Fast API response
* 🎨 Simple and clean UI using **Bootstrap**
* 🔐 API key secured using **environment variables**

# ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/priyanshubisht2001/weather-app.git
```

### 2️⃣ Go into the project folder

```bash
cd weather-app
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Create `.env` file

```
OPENWEATHER_API_KEY=your_api_key_here
```

You can get a free API key from:
https://openweathermap.org/api

### 5️⃣ Run the server

```bash
node app.js
```
